# Client React – JWT demo


## Instalar e rodar


# dentro de project-root/client
npm install
npm start


Abra http://localhost:3000


Fluxo de teste:
1. Informe username=testuser e password=testpass e clique Entrar.
2. Depois clique "Chamar /private" para recuperar a rota protegida.


Observação: o servidor deve estar rodando em http://localhost:4000 (ou ajuste `API_BASE` em src/App.js).
